What is the CSS Transition Property and what can it do for us?

The CSS Transition Property is applied to a specific element to make other proporties of the element transition smoothly over a given duration , or animate it in a nice way

____________________________________

Now onto the Transition-timing-function

This controls how the change in the property will occur

  • ease : the change starts slow and then goes faster, then finishes slow
  • linear : the change occurs at the same rate
  • ease-in : the change starts slow and then goes faster before the end
  • ease-out : the changes starts fast then slows down before the end
  • ease-in-out: the change starts slow then goes faster then finishes slow

There is also the Transition-Delay property which sets the time before the transition starts

it accepts values in seconds or milliseconds

You can also add more then one property to the transistion property CSS to effect more then one element

transition-property: background-color color width;

so each property seperated by a space, but if you need to apply the transition to alot, you can also put All

transition-property: all;

There is also a faster way, just by using "transition" you can pass multiple css options into one

transition: background-color 3s ease-in-out 1s, color 2s ease 4s;

Important to note, not all CSS properties can use CSS transitions since they are not animatable

  • background-image
  • display
  • border-style
  • position
  • float
  • background-repeat
  • font-family
  • and many others

Animated properties are those that can be expressed by numbers becasue the browser can interpurt the values such as width, height, border-width, padding, margin, opacity, etc

We can also animate properties that can be expressed by color such as the properties color and background-color
